# Weather_API_App
Project for experimentation with OpenWeatherMap API

## How to Run
1. Ensure dependencies are installed
   - `pip install geopy`
   - `pip install requests`
   - `pip install tzdata`
2. Create file "config.py" with contents:
   - `API_KEY="<YOUR_API_KEY_GOES_HERE>"`
3. run `python main.py`
